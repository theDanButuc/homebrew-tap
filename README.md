# Homebrew Tap for Claude Usage Monitor

A native macOS menu bar app that tracks your Claude.ai usage at a glance.

## Install

```bash
brew tap theDanButuc/tap
brew install --cask claude-usage-monitor
```

## Update

```bash
brew upgrade --cask claude-usage-monitor
```

## Uninstall

```bash
brew uninstall --cask claude-usage-monitor
```

## About

This tap provides the Homebrew Cask for [Claude Usage Monitor](https://github.com/theDanButuc/Claude-Usage-Monitor).

- Colour-coded menu bar icon (🟢 green → 🟡 yellow → 🔴 red)
- - Live usage counter in the menu bar
  - - Circular progress ring with reset timer
    - - No API key needed — just log in with your Claude.ai account
      - - Works with Free, Pro, Team, and Max plans
        - - Requires macOS 13 Ventura or later
